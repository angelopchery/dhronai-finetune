# DhronAI Installer

## Windows
Go to:
windows/dhronai-installer.exe

Double-click to install.

---

## Mac / Linux

Run:

cd unix
chmod +x install.sh
./install.sh

---

## After Installation

Activate environment:

Windows:
dhron_env\Scripts\activate

Mac/Linux:
source dhron_env/bin/activate

Then run:

dhronai --help
